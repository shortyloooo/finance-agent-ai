# Finance Agent AI

Finance Agent AI is a full-stack AI-powered personal finance tracking and budgeting system. It allows users to manage income and expenses, upload CSV transaction data, view financial analytics, and receive AI-generated budgeting advice.

## Tech Stack

- Python
- Streamlit
- Supabase
- LangGraph
- Groq
- Pandas

## Features

- User authentication
- Transaction management
- CSV transaction upload
- Income and expense dashboard
- Spending category breakdown
- Budget goal tracking
- AI-generated budgeting advice
